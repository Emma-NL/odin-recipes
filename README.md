In this project the goal was to create a homepage that directs to relative links with 3 different recipes. First of all it was important to set up the boilerplate for html for all pages. Next I had thought about which recipes I wanted. To support the recipes, I found images that were simmilar to the recipes i wanted, so I made an "images" folder and made sure the images were added to that folder in a png format. 

To tie everything together I used <a> to make links that direct to the recipe pages. To make the homepage look more exiting, I added the images as well using <img src>. 

The recipe pages are similar to each other. They all include a <h1>, <h2>, images and a link that directs the user back to the home page. I also added an ordered and unordered list to show the ingredients and steps. 

In short, in this project I have learned to direct relative links using <a>, use images using <img>, use typing features like <h1>, <h2> and <p>. Lastly I learned how to make unordered lists <ul> and ordered lists <ol> and made them visual using <li>.
